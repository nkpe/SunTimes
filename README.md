# SunTimes

Get sunset and sunrise times for locations using longitude and latitude, and optional date.

## Running the code locally

1. ```./gradlew run``` - starts the program. Follow prompts in command line.
2. Example input:
    * Latitude = 51.515972
    * Longitude = -0.174943
